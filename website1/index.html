PROJECTS:

Dynamic Sentence Input:

This C program, titled "Dynamic Sentence Input," allows users to input a complete sentence of variable length. The program dynamically allocates memory for the sentence, character by character, using calloc and realloc. It continuously accepts input until the user presses the Enter key ('\r'). The input is echoed to the console in real-time, demonstrating the dynamic memory allocation process. Once the user presses Enter, the program prints the entered sentence no matter how long it is.

Entry Test Management System:

This C program implements a Student Management System with a focus on test registration and administration. The system comprises an administrator interface for managing student records, checking test results, and appending questions to different tests (SAT, ECAT, Military). It also features a student interface for registration and taking various tests.

Banking System:

This C project implements a secure and user-friendly banking system allowing users to create accounts, check balances, transfer funds, and view transaction histories. Key features include user authentication, robust error handling, file-based data storage, and a modular design. The system is designed for educational purposes, emphasizing essential programming concepts. For real-world applications, additional security measures and a graphical user interface (GUI) could be incorporated.
